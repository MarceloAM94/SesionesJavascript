# SesionesJavaScript

Colección de ejercicios y ejemplos para aprender JavaScript, organizada por sesiones.

Estructura

- Sesion1/: Variables y tipos de datos
- Sesion2/: Operadores y estructuras de control
- Sesion3/: Funciones
- Sesion4/: Métodos de arrays

Cada carpeta contiene ejercicios prácticos (.js) y un `README.md` con la teoría y referencias.

Cómo ejecutar

1. Instalar Node.js (si no está instalado): https://nodejs.org/
2. Ejecutar un archivo con `node`:

```bash
node Sesion1/01-variables.js
```

Siguientes pasos sugeridos

- Añadir `package.json` con scripts para ejecutar ejemplos fácilmente.
- Añadir tests básicos para validar ejercicios.
- Publicar en GitHub Pages o crear documentación estática.
